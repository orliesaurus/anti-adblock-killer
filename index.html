How to add/update AakList ?

Add
Adblock: Options -> Filterlist -> Custom filter lists -> Enter a AakList URL and click "Subscribe"
Adblock Plus: Options -> Filter lists -> Add filter subscription -> Enter a AakList URL and click "Add"
uBlock Origin: Options -> 3rd-party-filters -> Enable AakList -> Click "Apply changes"
Update
Adblock: Options -> Filterlist -> Click "Update now"
Adblock Plus: Options -> Filter lists -> Click "Update now"
uBlock Origin: Options -> 3rd-party-filters -> Click "Update now"

Compatible

Adblock, Adblock Plus, uBlock Origin
